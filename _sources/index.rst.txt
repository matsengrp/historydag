historydag documentation
########################

``historydag`` is a package for representing collections of internally labeled
trees, or 'histories' in a directed acyclic graph. For examples and information
on getting started, see the quickstart below, or
`the historydag README on github <https://github.com/matsengrp/historydag>`_
